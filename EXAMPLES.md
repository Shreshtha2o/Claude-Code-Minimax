## Design Customization Tips

### Color Customization
```html
<!-- After copying HTML, find <style> section -->
<!-- Change primary color: -->
:root {
  --primary-color: #YOUR-COLOR; /* Change this */
}
```

### Text Customization
```html
<!-- Find and replace placeholder text -->
<h1>Your Business Name Here</h1>
<p>Replace with your actual message</p>
```

### Image Addition
```html
<!-- Add to your HTML -->
<img src="your-image-url.jpg" alt="Description" style="width: 100%;">
```

### Font Changes
```css
/* In <style> section */
body {
  font-family: 'Your Font', sans-serif; /* Change font family */
}
```

## Best Practices for Each Platform

### Instagram
- **Dimensions**: 1080x1350px (Portrait)
- **Best Content**: Lifestyle, product showcase, behind-the-scenes
- **Frequency**: 3-5 posts per week
- **Captions**: Storytelling, emoji use, hashtags
- **CTAs**: Save, share, link in bio

### Facebook
- **Dimensions**: 1200x628px (Landscape)
- **Best Content**: Community building, promotions, education
- **Frequency**: 1-2 posts daily
- **Captions**: Conversational, engaging questions
- **CTAs**: Like, comment, share, visit website

### LinkedIn
- **Dimensions**: 1200x627px (Landscape)
- **Best Content**: Professional insights, company updates, thought leadership
- **Frequency**: 2-3 times per week
- **Captions**: Professional but personable
- **CTAs**: Congratulations, support, visit page

### Twitter/X
- **Dimensions**: 1024x512px (Landscape)
- **Best Content**: News, quick insights, engagement
- **Frequency**: 2-5 times daily
- **Captions**: Concise, witty, timely
- **CTAs**: Reply, retweet, like

### TikTok
- **Dimensions**: 1080x1920px (Portrait, vertical)
- **Best Content**: Trends, entertainment, behind-the-scenes
- **Frequency**: 1-3 times daily (if video)
- **Captions**: Trending sounds, hashtags
- **CTAs**: Like, follow, duet, stitch

### Pinterest
- **Dimensions**: 1000x1500px (Portrait)
- **Best Content**: Inspiration, DIY, products, ideas
- **Frequency**: 5-15 pins daily
- **Captions**: Keywords, CTAs, descriptions
- **CTAs**: Save, click through, visit website

### YouTube
- **Dimensions**: 1280x720px (Landscape, 16:9)
- **Best Content**: Education, entertainment, tutorials
- **Frequency**: 1-3 videos per week
- **Captions**: Detailed descriptions, tags
- **CTAs**: Subscribe, like, comment, visit website

## Export & Usage Guide

### Download HTML
1. Click "Download HTML" button
2. File saves as `BusinessName-Platform-Timestamp.html`
3. Open in any browser to view
4. Edit in code editor (VS Code, Notepad++, etc.)

### Copy for Email
1. Click "Copy Code" button
2. Paste into email builder
3. Most platforms accept HTML
4. Test rendering in email client

### Use with Design Tools
1. Export HTML
2. Screenshot or print to PDF
3. Import into Canva, Figma, etc.
4. Further customize as needed

### Share with Team
1. Download HTML file
2. Share via email, Slack, or cloud storage
3. Team can open and edit
4. Collaborate on customizations

## Troubleshooting Generated Designs

| Issue | Solution |
|-------|----------|
| Colors look different | Check browser color settings, try different browser |
| Text is hard to read | Verify contrast ratio, adjust in code |
| Design doesn't fit platform | Check dimensions, crop or resize in code |
| Images not showing | Add image URLs to HTML |
| Layout looks broken | Open in different browser, check CSS |
| Can't edit HTML | Use plain text editor, not Word |

## Advanced Customization Examples

### Add Gradients
```css

```

### Add Animations
```css
animation: fadeIn 1s ease-in;
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
```

### Add Hover Effects
```css
button:hover {
  transform: scale(1.05);
  box-shadow: 0 5px 15px rgba(0,0,0,0.3);
}
```

### Add Responsive Layout
```css
@media (max-width: 768px) {
  .container {
    padding: 10px;
  }
}
```
