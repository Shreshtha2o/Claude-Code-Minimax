## 🚀 Quick Start (5 Minutes)

### 1. Prerequisites
```bash
# Check Node.js
node --version  # v14 or higher

# Check npm
npm --version
```

### 2. Installation
```bash
cd social-media-design-ai
npm install
```

### 3. Configure
```bash
cp .env.example .env
# Edit .env and add your CLAUDE_API_KEY
```

### 4. Run
```bash
# Terminal 1
npm start

# Terminal 2
npm run frontend
```

### 5. Access
Visit: **http://localhost:8000**
